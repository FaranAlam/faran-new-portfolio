## 🎨 Design Philosophy

### 1. **Subtle, Not Overwhelming:**
- Loading screen: 2 seconds only (not annoying)
- Progress bar: Thin line (1px, not intrusive)
- Particles: 30% opacity (background, not foreground)
- Typing effect: Natural speed (readable)
- Stats ticker: Semi-transparent (doesn't block content)

### 2. **Performance Optimized:**
- Canvas particles: GPU accelerated (60fps smooth)
- Loading screen: Exits DOM after animation (no memory leak)
- Progress bar: CSS only (no heavy JS)
- Typing effect: Lightweight (< 1KB)
- Stats ticker: Conditional render (only when needed)

### 3. **Mobile-First:**
- Loading screen: Scales to mobile
- Progress bar: Works on touch devices
- Particles: Reduced count on mobile (performance)
- Typing effect: Readable on small screens
- Stats ticker: Marquee animation on mobile
