--- slide3.xml ---
In Android, an  explicit intent  is an intent that clearly specifies the target component (such as a particular  Activity ,  Service , or  BroadcastReceiver ) that should handle the request. In simple words: With an  explicit intent , you tell exactly which component you want to open. This is mostly used  inside your own app , when you want to move from one activity to another or start a known service. Use Case: Navigate from one screen (Activity) to another. Pass data between activities. What are Explicit Intents? 3

To open  Activity_Second  from  MainActivity , you can use an explicit intent: Here: this  refers to the current context ( MainActivity ). Activity_Second ::class.java  tells Android exactly which activity to open.
