# Public Assets

This folder contains static assets like:
- Images
- Fonts
- Icons
- Other static files

## Folder Structure

```
public/
├── images/
│   ├── portfolio/
│   ├── blog/
│   └── profile/
└── favicon.ico
```

Add your static files to this folder and reference them in your components like:

```jsx
<Image src="/images/profile/avatar.jpg" alt="Profile" />
```
